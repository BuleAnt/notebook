使用git来管理 Azkaban的工作：

​	当出现提交时触发打包脚步，将Azkaban的作业打包





=========================================

```
- azgz init
// 在当前目录执行 git init, 创建git仓库

- azgz clone
// 在当前目录下创建git仓库

azgz add
// 添加用户信息

azgz commit -m ""
// 添加描述

azgz 
```
